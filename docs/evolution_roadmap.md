## Batch added 2026-03-12

> Source: online_search — trending features for AI developer productivity tools 2026

### Context-Aware Code Review Agents

**Status:** Drafted

AI agents that automatically review PRs by understanding the full project context — architecture, past decisions, and team conventions — rather than reviewing diffs in isolation. Reduces review fatigue and catches deeper semantic issues.

*Added: 2026-03-12*

### Ambient Pair-Programming Copilot

**Status:** Drafted

A persistent background agent that observes the developer's workflow (editor actions, terminal output, test failures) and proactively suggests the next relevant action, snippet, or documentation link without needing an explicit prompt.

*Added: 2026-03-12*

### Automated Knowledge-Gap Flashcards

**Status:** Drafted

After analysing learning materials and detecting knowledge gaps, auto-generate spaced-repetition flashcards (Anki / Mochi format) so developers actively reinforce weak areas discovered during the analysis pipeline.

*Added: 2026-03-12*
